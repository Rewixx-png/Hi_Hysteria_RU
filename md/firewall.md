#### Настройка FireWall (Брандмауэра)

Скрипт `hihy` пытается автоматически открыть порты через `iptables` / `ufw`.
Однако, на многих облачных хостингах (Oracle Cloud, AWS, Alibaba Cloud, Google Cloud) есть **внешний фаервол** в панели управления сайтом.

**Что нужно открыть:**

1.  **UDP порт сервера** (тот, который вы указали при установке, по умолчанию 443).
2.  **TCP порт 80** (Если используете автоматический сертификат ACME/Let's Encrypt).
3.  **Диапазон портов (Port Hopping)**: Если вы включили эту функцию (например, 20000-50000), этот диапазон UDP портов должен быть открыт!

**Если не подключается:**
В 90% случаев проблема именно в закрытых портах на стороне хостинга. Зайдите в панель управления VPS -> Security List / Firewall и откройте `UDP 0-65535` (или конкретные порты).
